# Live2D Mao Model Placeholder

`/public/live2d/models/mao` ディレクトリには、Live2D Cubism SDK for Web に付属する **Mao** モデル一式を配置してください。

必要なファイル例:

- `mao_pro_t03.model3.json`
- `mao_pro_t03.moc3`
- `mao_pro_t03.cdi3.json`
- `mao_pro_t03.physics3.json`
- `mao_pro_t03.pose3.json`
- `expressions/*.exp3.json`
- `motions/*.motion3.json`
- `textures/` や `mao_pro_t03.2048/texture_00.png` などのテクスチャ画像

モデルは Live2D 公式の Cubism SDK for Web に含まれるサンプル (`Samples/Resources/Mao/`) から取得できます。ファイルを配置すると、`Character` 画面の Live2D 表示が有効になります。
